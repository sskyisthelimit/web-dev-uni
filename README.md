Creator: Yalanskyi Roman — Group KI-32

Short description
A small static portfolio/gallery site built from a Figma template.

Project structure
- index.html — landing / splash page
- home.html — main site page with newsletter and featured images
- about.html — about the artist
- gallery.html — image gallery (uses lightgallery via CDN)
- shop.html — items / paintings for sale
- contact.html — contact form and contact details
- css/ — stylesheets (style.min.css, lightgallery css)
- js/ — JavaScript (scripts.min.js, lightgallery)
- img/ — images and icons used across pages

How to run / preview
- Open index.html (or any page) in a browser.

Notes
- gallery.html loads lightgallery from a CDN; internet connection required for that feature.
